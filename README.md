# 🚀 Modern Login & Signup System with Supabase Auth

A sleek, premium, and fully responsive user authentication system built using modern frontend technologies and **Supabase** backend. This project showcases a clean User Interface (UI) with interactive animations, real-time validations, and secure authentication flows.

🌐 **Live Demo:** [Click Here to View Live App](https://shahrozkhan-tech.github.io/Login-system-with-supabase/)

---

## ✨ Features

- **🔐 Secure Authentication:** Full Login and Signup flow powered by Supabase Auth.
- **📧 Email Verification:** Automated secure magic link/verification email sent upon signup.
- **🎨 Next-Level UI:** Modern design with glassmorphism touches, smooth transitions, and dynamic visual states.
- **📱 100% Responsive:** Optimized for all screen sizes from mobile devices to desktop monitors.
- **🔔 Interactive Feedbacks:** Integrated popup notifications for success, warnings, and error handlings.
- **🔄 Protected Dashboard:** Dynamic redirection to a secure area after successful authentication.

---

## 🛠️ Tech Stack Used

- **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid, Animations), JavaScript (ES6+)
- **Backend-as-a-Service (BaaS):** Supabase (PostgreSQL, GoTrue Auth)
